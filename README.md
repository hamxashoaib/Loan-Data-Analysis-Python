# Loan Approval Prediction Project

I’m excited to share my Loan Approval Prediction project, where I analyzed and predicted loan approval status using Python in Google Colab. This project helped me strengthen my skills in data cleaning, exploratory analysis, visualization, and basic machine learning.

## Project Overview
This project uses a real-world loan applicant dataset to uncover patterns and predict whether a loan will be approved. The main goal was to clean the data, explore important variables, and build a simple predictive model to identify trends.

## Key Steps Performed
- Loaded and explored the dataset using Pandas  
- Handled missing values using mean/mode imputation  
- Removed duplicate and inconsistent entries  
- Identified numerical and categorical features  
- Performed statistical analysis (mean, min, max, standard deviation)  
- Encoded categorical variables for modeling  
- Split the dataset into training and testing sets  
- Applied basic machine learning models to predict loan approval  

## Data Visualization
Used Matplotlib and Seaborn to create visualizations:  
- **Bar Charts** – to compare loan approval rates across categories  
- **Scatter Plots** – to analyze relationships (e.g., income vs loan amount)  
- **Histograms** – to understand data distribution  
- **Box Plots** – to detect outliers  
- **Count Plots** – to explore categorical variable distributions  

## Key Insights
- Credit history strongly impacts loan approval  
- Higher applicant income generally increases approval chances  
- Certain features have a stronger influence on model predictions
